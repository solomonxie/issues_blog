U-substitution ‚Üí Chain Rule

The u-substitution is to solve an integral of composite function, which is actually to UNDO the Chain Rule.

Compare how we handle the composite functions with derivatives & integrals: - For taking the derivative of a COMPOSITE function, we apply the Chain rule. - For taking the integral of a COMPOSITE function, we apply the u-substitution.
