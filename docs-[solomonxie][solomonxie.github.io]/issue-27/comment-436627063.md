# Linux 用户管理


## 添加新Admin用户

[参考：How To Create a Sudo User on Ubuntu ](https://www.digitalocean.com/community/tutorials/how-to-create-a-sudo-user-on-ubuntu-quickstart)

```sh
# 添加用户
$ sudo adduser test

# 根据交互，填写密码和默认信息
#...

# 将用户添加到sudoer权限组
$ sudo usermod -aG sudo test

# 切换到用户
$ su - test
```
