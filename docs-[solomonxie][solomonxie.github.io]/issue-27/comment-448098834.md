# ❖ Linux 搜索工具 [DRAFT]

## 文件搜索
- find
- fd: 实现find80%功能，速度更快，增加忽略文件功能


Mac安装：
```sh
brew install fd
```

Ubuntu安装：
```sh
sudo apt-get install fd-find
```
