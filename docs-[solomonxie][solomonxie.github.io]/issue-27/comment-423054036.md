## 常用

假设我们用`ping -c 10 localhost >> ping.txt`文件，然后用sed来操作：

```
# 只显示第1行
$ cat ping.txt | sed -n '1p'
# 只显示最后一行(这个无需sed)
$ tail ping.txt -n 1

# 只显示第2-4行
$ cat ping.txt | sed -n '2,4p'

# 删除第2-4行并显示
$ cat ping.txt | sed '2,4d'
```
