# Linux 路径名称操作
Linux自带的`basename`和`dirname`命令就极其好用，很多情况都不需要`sed`和`awk`等高级复杂工具。

```sh
# 获取路径名
$ dirname "/etc/apt/sources.list"
>>> /etc/apt

# 获取当前脚本的绝对路径
$ dirname $0

# 获取当前脚本的父目录的绝对路径
$ dirname $(pwd)

$ dirname "/etc/apt/"
>>> /etc

# 获取文件名（或最后一个目录名）
$ basename "/etc/apt/sources.list"
>>> sources.lit

$ basename "/etc/apt/"
>>> apt

# 获取名称不包括扩展名
# (通过去掉结尾的指定文字来达到)
$ basename "/etc/apt/sources.list" .list
>>> sources
```
