## Axel 超强超速多线程的下载器 代替wget

Mac安装：
```sh
brew install axel
```

Ubuntu安装：
```sh
sudo apt-get install axel
```

使用：
```sh
$ axel <URL>
```

指定线程数：
```sh
$ axel -n 10 <URL>
```
