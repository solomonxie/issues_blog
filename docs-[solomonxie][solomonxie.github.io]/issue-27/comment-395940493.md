# Linux的cp, mv, rm等显示当前操作或进度的方法

## 拷贝或移动文件夹时`/`的用法
假设现有一个source文件夹:
```
source
    sub
        a.jpg
    b.jpg
    c.jpg
```

以cp为例，其它mv、scp、rsync等都一样：
```sh
$ cp  -r source target
如果target目录不存在，则target与source为完全相同的目录情况：
target
    ......
source
    ......

$ cp  -r source target
如果target目录已存在，且为空目录，则在target目录下存放source目录：
target
    source
        ......
```


## mv
```sh
# -v参数verbose
$ mv -v ./src/* /tmp
```

## rm
```sh
# -v 和-r参数 注意文件列表太多的话会不执行
$ rm -vr <from> <to>
```

## cp
```sh
# -v 和-r参数 verbose & recursive
$ cp -vr <from> <to>

# 使用rsync命令代替
$ rsync -avP <FROM> <TO>
```
