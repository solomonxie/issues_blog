# Disk Management 磁盘管理


## Badblocks 坏道修复

坏道检查命令`/sbin/badblocks`：
```sh
$ sudo badblocks /dev/sda
```

cron任务定期坏道检查，编辑`/etc/cron.d/badblocks`文件：
```sh
30 4 * * 3 root [ -x /sbin/badblocks ] && [ $(date +\%d) -le 7 ] && /sbin/badblocks /dev/sda
```



## 空白硬盘分区、格式化

```sh
# 查看添加的网盘名称和地址(如/dev/sda)
$ lsblk

# 查看网盘的文件系统状况（是否格式化了）
$ sudo file -s /dev/sda
```
