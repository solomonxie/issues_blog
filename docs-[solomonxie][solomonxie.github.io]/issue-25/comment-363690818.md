# Vim自动运行（或编译运行）文件
不像SublimeRepl需要安装插件，Vim原生支持直接运行python等代码。只要输入命令：
`:!python %`
其它语言代码类似。
输入命令后，vim界面会暂时跳出编辑页面跳到终端页面显示执行过程，期间可以按任意键返回vim编辑页面。
输过一次后，就可以用`:!!`直接重复上次的命令，不用再输一遍命令。
