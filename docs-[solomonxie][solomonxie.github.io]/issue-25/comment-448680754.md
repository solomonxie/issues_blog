## Functions 函数

```vim
function MyFunction()
    :wq
endfunction
```

但是如果重载当前vimrc的话，会遇到`function already exists`警告。
所以最好在将函数定义为可重写的函数，即变为`function!`：
```vim

```vim
function! MyFunction()
    :wq
endfunction
```
