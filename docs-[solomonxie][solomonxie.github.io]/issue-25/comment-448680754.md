## 逻辑控制

if-else:
```vim
if 3 >= 1
    echo 'True'
endif
```

VIM中的字符串和数字是可以直接比较的，如`:echo '3' >= 3`，返回1.


## Functions 函数

```vim
function MyFunction()
    :wq
endfunction
```

但是如果重载当前vimrc的话，会遇到`function already exists`警告。
所以最好在将函数定义为可重写的函数，即变为`function!`：
```vim

```vim
function! MyFunction()
    :wq
endfunction
```
