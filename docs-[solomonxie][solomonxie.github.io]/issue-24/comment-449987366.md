## 递归+yield

[参考：Recursion using yield](https://stackoverflow.com/questions/8991840/recursion-using-yield)

```py
class MyClass:

    def recursion(self, num):
        yield num
        if num < 100:
            yield from self.recursion(num+1)

obj = MyClass()
for  i in obj.recursion(100):
    print( i )
```
