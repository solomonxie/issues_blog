# Python装饰器

极简样式：
```py
# Define a decorator
def custommethod(func):
    def method_code():
        # ...Some code here...
        func()
        # ...Some code here...

    return method_code

# Apply a decorator
@custommethod
def stable_method():
    pass
```

标准样式：
```py
# Define a decorator
def custommethod(func):
    def method_code(*args, **kargs):
        # ...Some code here...
        func(*args, *kargs)
        # ...Some code here...

    return method_code

# Apply a decorator
@custommethod
def stable_method():
    pass
```

装饰器带参数样式：
```py
# Define a decorator
def outter(name):
    def inner(func):
        def method_code(*args, **kargs):
            # ...Some code here...
            func(*args, *kargs)
            # ...Some code here...
            print('I can use "name={}" here.".format(name))

    return method_code

# Apply a decorator
@custommethod(name='Jason')
def stable_method():
    pass
``
