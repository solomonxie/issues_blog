# Python装饰器

极简样式：
```py
# Define a decorator
def outter(func):
    def alternative():
        # ...Some code here...
        return func()
        # ...Some code here...

    return alternative

# Apply a decorator
@outter
def stable_method():
    pass
```

标准样式：
```py
# Define a decorator
def outter(func):
    def alternative(*args, **kargs):
        # ...Some code here...
        return func(*args, **kargs)
        # ...Some code here...

    return alternative

# Apply a decorator
@outter
def stable_method():
    pass
```

装饰器带参数样式：
```py
# Define a decorator
def outter(name):
    def inner(func):
        def alternative(*args, **kargs):
            print('I can use name={} here'.format(name))
            # ...Some code here...
            return func(*args, **kargs)
            # ...Some code here...

        return alternative
    return inner

# Apply a decorator
@outter(name='Jason')
def stable_method():
    pass
``
