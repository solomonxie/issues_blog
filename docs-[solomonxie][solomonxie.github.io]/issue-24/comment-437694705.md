# Python装饰器

基本样式：
```py
# Define a decorator
def custommethod(func):
    def method_code():
        # ...Some code here...
        func()
        # ...Some code here...

    return method_code

# Apply a decorator
@custommethod
def stable_method():
    pass
```

带参数样式：
```py
# Define a decorator
def custommethod(func):
    def method_code(*args, **kargs):
        # ...Some code here...
        func(*args, *kargs)
        # ...Some code here...

    return method_code

# Apply a decorator
@custommethod
def stable_method():
    pass
```
