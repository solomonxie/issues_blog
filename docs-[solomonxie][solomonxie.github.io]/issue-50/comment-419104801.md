# `Hypothesis Testing`

## `H₀ & Ha`
- H₀: Null Hypothesis
- Ha: Alternative Hypothesis

## `Significance Level ⍺` (Threshold)

- 1%
- `5%`
- 10%


## `p-value`
> p-value stands for _Probability value_.

p-value is a **`Conditional Probability`**.
