# `Z or T`?

## Notations
- Standardized Distribution:
    - Normal distribution -> Z-score: `z`
    - T distribution -> T-score: `t`
- Confidence Interval
    - Z interval: `z*` or `z-value`
    - T interval: `t*` or `t-value`
- Hypothesis Test
    - Z statistic: `z` or `z-value`
    - T statistic: `t` or `t-value`
