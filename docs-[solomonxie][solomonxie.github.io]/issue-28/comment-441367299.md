# MongoDB Indexing 索引

## 速度查询

显示某命令的执行效率：
```js
db.collection1.find().explain('executionStats');
```
然后会返回一个非常详尽的执行信息，其中的`ExecutionTimeMillis`代表语句的执行时间，`Millis`代表`milliseconds`毫秒。
