## 理解Scrapy流程

`Spider` -> `item` -> `Pipline`

其中：
- `Spider`只负责向目标网站，获得response，传给item
- `item`只负责从response中整理数据，变成item字典对象，传给pipline
- `Pipline`只负责把item字典对象转换为ORM，保存到数据库
