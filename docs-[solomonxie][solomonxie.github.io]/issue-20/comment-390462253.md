# Experiment: 心灵感知，五官感知 @2008-3-27
心灵感知，五官感知…
有时我们戴上耳机听到绝妙的音乐，也想旁边的人能感受到。但这不可能做到，因为一副耳机只能戴一对耳朵上

于是我尝试耳朵感知的能力。

在一个非常封闭的方间里，我给一个人戴上耳机并让他安静坐在那，我对着他后背，开始尝试进入他的听觉系统。因为对方是不动的，所以搜索到那个波动不算太难

接着我开始进行提高练习：
让他走动，然后我去感知…这个就比较难了，

怎么办呢，波动的方位一直都在变我不能专心并快速捕捉还有不断切换，
就像调频收音机，如果音乐之声的频道一直在变动你怎么办呢？

是让自己变快还是有其它什么方法？
