# WSGI理解

WSGI (Web Server Gateway Interface)，是一个**`单向`**的程序：客户端->服务器。

在客户端向服务器发出请求时，HTTP Server服务器接收请求，进行socket、tcp、udp等基层操作，全部验证通过后，再把请求内容发给WSGI。然后WSGI作为一个Distributor分发者，把请求发给服务器上具体的某个app应用，比如Wordpress，或Django。
