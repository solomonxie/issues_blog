# SSL证书

## Terms
`HTTPS`是在HTTP上加了一层Layer，Secure Layer，所以是HTTPS。而这个Layer，叫SSL(ayer)。
实现这个Layer，需要一个证书，就叫SSL 证书，是CA证书的一种。
`CA证书`泛指一切Key-pair数字证书对，即公钥和私钥。而遵循SSL协议的CA证书，就是SSL证书了。
而CA证书包括这几种：
- `DV`，针对个人的证书
- `OV`，针对企业的证书，需要对企业资格审核，安全
- `EV`，针对企业的增强型证书，审核最严格，最安全
