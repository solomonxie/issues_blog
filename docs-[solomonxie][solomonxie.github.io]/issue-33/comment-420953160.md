# `Virtualbox连接网络`

[参考：快速理解VirtualBox的四种网络连接方式](https://www.cnblogs.com/york-hust/archive/2012/03/29/2422911.html)

Virtualbox连接网络一共有四种方式（其它虚拟机也都差不多一样的称呼）:
- NAT
- Bridged Adapter
- Internal
- Host-only Adapter
