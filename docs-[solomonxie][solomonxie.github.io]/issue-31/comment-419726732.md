## Windows上远程连接另一台Windows桌面
Windows有内置的远程基础设施，所以可以远程、客户端都可以无需安装任何工具就能连接。

Windows服务端设置：
- 开始菜单中搜索`远程连接` 
- 打开设置中的远程连接设置 
- 允许远程连接等设置。
- 为机器设置用户名密码（不能为空密码）

同一时间只能在一台电脑中使用同一个用户，如果远程是用administrator用户登录的，那么本地的administrator用户就会被弹出。


Windows客户端设置：
开始菜单中搜索`远程连接` - 选择远程连接 - 输入服务端的IP地址和用户名 - 忽略证书之类的直接登录。
