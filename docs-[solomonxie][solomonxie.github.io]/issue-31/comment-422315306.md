# VPS分区测速

主要方法：
```sh
# SSH 连接
$ time ssh ubuntu@IP exit

# 上传下载文件（文件是10K)
$ time scp ~/scptest.jpg ubuntu@IP:~/
$ time scp ubuntu@IP:~/scptest.jpg ~/

# Ping youtube
$ ssh ubuntu@IP "ping -c 1 youtube.com ; exit"
```
