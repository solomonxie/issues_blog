# VPS分区测速

主要方法：
```sh
# 直接Ping
$ ping -c 10 IP:Port 

# SSH 连接
$ time ssh ubuntu@IP exit

# 上传下载文件（文件是1M)
$ time scp ~/scptest ubuntu@IP:~/
$ time scp ubuntu@IP:~/scptest ~/
```
