# `AWS S3 静态云存储`


## 创建Bucket
登录AWS进入S3 -> create bucket -> 创建S3全网唯一名称 -> 默认选项 -> 选择公开数据的浏览权限 -> 完成创建Bucket。

之后都可以修改，填错了也没问题。
