开始前，你需要知道的一切：
```sh
$ dnf install podman -y
$ alias docker=podman
```

## 什么是Podman

能够快速增强理解的说法是，
> `Podman`是`Docker-compose`的替代品。

也就是说，`podman`用来管理多个相关联的Docker容器，达到比docker-compose更好更合理的配合。
